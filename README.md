# OpenClaw Skills Collection

这是一个自定义的 OpenClaw 技能集合，包含多个实用的自动化技能，涵盖了地图导航、数据分析、信息爬取、办公自动化等多个领域。

## 技能列表

### 1. 高德地图路线规划 (`amap-route`)
- **功能**：基于高德地图 API 的路线规划，支持驾车、步行、公交、骑行四种出行模式
- **使用场景**：查询两地之间的路线、距离、耗时、导航方案
- **触发词**：路线规划、怎么走、怎么去、导航、多远、多久、驾车路线、公交路线、步行路线、骑行路线

### 2. 中国新闻爬虫 (`china-news-crawler`)
- **功能**：爬取中国主要新闻网站的新闻内容
- **使用场景**：获取最新新闻资讯、批量收集新闻数据

### 3. Excel 数据处理 (`excel-xlsx`)
- **功能**：Excel 文件的读取、写入和处理
- **使用场景**：数据分析、数据转换、报表生成

### 4. 飞书日历 (`feishu-calendar`)
- **功能**：飞书日历的操作和管理
- **使用场景**：日历事件管理、日程安排

### 5. 技能查找 (`find-skills`)
- **功能**：查找和搜索可用的 OpenClaw 技能
- **使用场景**：技能发现、快速定位所需功能

### 6. GitHub 趋势 (`github-trending`)
- **功能**：获取 GitHub 上的趋势仓库和开发者信息
- **使用场景**：技术趋势分析、开源项目发现

### 7. Hugging Face 趋势 (`huggingface-trending`)
- **功能**：获取 Hugging Face 上的趋势模型和数据集
- **使用场景**：AI 模型发现、自然语言处理资源查找

### 8. PDF 处理 (`pdf`)
- **功能**：PDF 文件的读取、解析和处理
- **使用场景**：文档处理、数据提取、PDF 转换

### 9. 自我改进 (`self-improvement`)
- **功能**：个人学习和自我提升相关的功能
- **使用场景**：学习计划、知识管理

### 10. 发送邮件 (`send-email`)
- **功能**：发送电子邮件
- **使用场景**：自动化通知、报告发送

## 技能结构

每个技能都遵循以下结构：
```
skill-name/
├── SKILL.md          # 技能描述和使用说明
├── scripts/          # 脚本文件
├── references/       # 参考资料
├── .clawhub/         # OpenClaw 相关配置
├── memory-template.md # 记忆模板
├── _meta.json        # 元数据
└── setup.md          # 设置说明
```

## 使用方法

1. **安装 OpenClaw**（如果尚未安装）
2. 导航到技能目录
3. 查看该技能的 `SKILL.md` 文件以获取详细使用说明
4. 按照说明运行相应的脚本或命令

## 开发说明

### 创建新技能

要创建新的 OpenClaw 技能，请遵循以下步骤：

1. 创建新的技能目录
2. 在目录中创建 `SKILL.md` 文件，包含技能的名称和描述
3. 实现技能的脚本或功能
4. 创建必要的配置文件和文档
5. 测试技能功能

### 技能描述格式

在 `SKILL.md` 文件中，使用以下格式：

```markdown
---
name: skill-name
description: 技能的详细描述，包括功能、使用场景、触发词等
---

# 技能名称 Skill

## 使用流程
1. 步骤1
2. 步骤2
3. 步骤3

## 运行脚本
```bash
command-to-run
```

## 注意事项
- 注意事项1
- 注意事项2
```
